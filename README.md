# Weekly GitHub Dev Summary — n8n + Claude Workflow

**Bounty #5** | $200 | claude-builders-bounty

## Overview

Automated weekly GitHub activity summaries generated by Claude (Sonnet 4) via n8n workflow. Fetches commits, closed issues, and merged PRs for the past week, then uses Claude API to produce a narrative report delivered via Discord or Email.

## Features

- ⏰ **Weekly cron trigger** (Friday 5pm, configurable)
- 📊 **GitHub API**: commits, closed issues, merged PRs
- 🤖 **Claude API**: generates narrative summary
- 📨 **Dual delivery**: Discord webhook OR Email
- 🌐 **Multi-language**: EN/FR/ZH configurable
- 🔧 **Configurable via environment variables** — no hardcoding

## Setup (5 steps)

### 1. Install n8n
```bash
npm install -g n8n
# or docker: docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
```

### 2. Import workflow
Open n8n → Workflows → Import from File → select `weekly-dev-summary-v2.json`

### 3. Configure environment variables
In n8n Variables, set:

| Variable | Value |
|----------|-------|
| `REPO` | `owner/repo` (e.g. `claude-builders-bounty/claude-builders-bounty`) |
| `GITHUB_TOKEN` | `ghp_...` (GitHub PAT with repo read access) |
| `ANTHROPIC_API_KEY` | `sk-ant-...` |
| `DELIVERY_METHOD` | `discord` or `email` |
| `DISCORD_WEBHOOK_URL` | `https://discord.com/api/webhooks/...` |
| `EMAIL_TO` | `your@email.com` |
| `LANGUAGE` | `EN` (or `FR`, `ZH`) |

### 4. Configure schedule
The trigger defaults to Friday 5pm (`0 17 * * 5`). Edit the Cron expression in the Schedule Trigger node to change.

### 5. Activate
Toggle workflow **Active** in n8n. Click "Test workflow" to run manually.

## Architecture

```
Schedule (Friday 5pm)
  ├── Fetch Commits ──────────┐
  ├── Fetch Closed Issues ───┤→ Compile Activity → Claude API → Route Delivery → Discord/Email
  └── Fetch Merged PRs ──────┘
```
